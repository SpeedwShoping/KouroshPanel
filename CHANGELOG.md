# Changelog — تاریخچه تغییرات

All notable changes to **Kourosh Panel** are documented here.

## [1.0.0] — 2026-07-20

### 👑 First official release — اولین نسخه رسمی

- 🎨 **Achaemenid design**: imperial gold & Persian turquoise palette, animated grid backdrop, glassmorphism cards, Faravahar logo — دیزاین هخامنشی با پالت طلایی/فیروزه‌ای، گرید انیمیشنی و کارت‌های شیشه‌ای
- 📜 **Quote of the visit**: the subscription page shows a fresh quote from Cyrus the Great or Reza Shah Pahlavi (FA/EN) on every visit — نمایش سخن جدید از کوروش کبیر و رضاشاه در هر بازدید صفحه ساب
- 📅 **Jalali calendar by default** + full Persian UI with the Vazirmatn font — تقویم شمسی پیش‌فرض و فارسی کامل
- 🖥️ **Professional CLI**: sectioned gold/turquoise terminal menu via the `kourosh` command — منوی ترمینال حرفه‌ای
- 📊 Animated statistics dashboard — داشبورد آماری انیمیشن‌دار
- 🔒 Full Xray feature set: VLESS, VMess, Trojan, Shadowsocks, WireGuard, Hysteria, MTProto; Reality, TLS, gRPC, WS, XHTTP
- 🤖 Telegram bot, notifications, automatic backups
- 🐳 Docker (ghcr.io/speedwshoping/kouroshpanel) & PostgreSQL support
