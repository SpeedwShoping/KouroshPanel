## 🇮🇷 فارسی

### درباره پروژه

**پنل کوروش** یک پنل مدیریت کامل Xray است با تمام امکانات حرفه‌ای، طراحی‌شده با هویت بصری ایرانی و هخامنشی — رنگ‌های طلایی و فیروزه‌ای، پس‌زمینه گرید انیمیشنی و کارت‌های شیشه‌ای.

### ✨ امکانات

- 🎨 **دیزاین هخامنشی مدرن** — تم طلایی/فیروزه‌ای با انیمیشن، حالت تاریک/روشن/فوق‌تاریک
- 📜 **سخن بزرگان در صفحه ساب** — در هر بازدید از لینک سابسکریپشن، یک سخن جدید از کوروش کبیر و رضاشاه پهلوی نمایش داده می‌شود
- 📅 **تقویم شمسی (جلالی) پیش‌فرض** + رابط کاربری کامل فارسی با فونت وزیرمتن
- 📊 **داشبورد آماری زنده** — نمودار مصرف، وضعیت CPU/RAM، ترافیک لحظه‌ای
- 🔒 پشتیبانی از تمام پروتکل‌ها: VLESS, VMess, Trojan, Shadowsocks, WireGuard, Hysteria, MTProto و...
- 🌐 Reality, TLS, gRPC, WebSocket, XHTTP و تمام ترنسپورت‌ها
- 👥 مدیریت کاربران، گروه‌ها، محدودیت حجم/زمان/IP
- 🔗 لینک سابسکریپشن (عادی / JSON / Clash) با صفحه اختصاصی لوکس
- 🤖 ربات تلگرام، نوتیفیکیشن، بکاپ خودکار
- 🖥️ منوی ترمینال حرفه‌ای رنگی با دستور `kourosh`
- 🐳 پشتیبانی Docker و PostgreSQL

## 🇬🇧 English

### About

**Kourosh Panel** is a full-featured Xray management panel with a unique Persian/Achaemenid visual identity — imperial gold and turquoise, animated grid backdrop, and glassmorphism cards.

### ✨ Features

- 🎨 **Modern Achaemenid design** — gold/turquoise theme, dark / light / ultra-dark modes
- 📜 **Quote of the visit** — the subscription page shows a fresh quote from Cyrus the Great or Reza Shah Pahlavi on every visit (FA/EN)
- 📅 **Jalali (Persian) calendar by default** + full Persian UI with the Vazirmatn font
- 📊 **Live statistics dashboard** — usage charts, CPU/RAM gauges, realtime traffic
- 🔒 All protocols: VLESS, VMess, Trojan, Shadowsocks, WireGuard, Hysteria, MTProto, …
- 🌐 Reality, TLS, gRPC, WebSocket, XHTTP and all transports
- 👥 Client & group management, traffic / expiry / IP limits
- 🔗 Subscription links (plain / JSON / Clash) with a luxurious dedicated page
- 🤖 Telegram bot, notifications, automatic backups
- 🖥️ Professional colorized terminal menu via the `kourosh` command
- 🐳 Docker & PostgreSQL support

### 📥 Install

```bash
bash <(curl -Ls https://raw.githubusercontent.com/SpeedwShoping/KouroshPanel/main/install.sh)
```

Manage the panel any time by running `kourosh` in your terminal.
